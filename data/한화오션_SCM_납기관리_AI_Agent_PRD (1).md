## 1. Executive Summary

### 1.1 배경
한화오션 상선 SCM팀은 현재 수작업으로 납기 관리를 수행하고 있으며, PO(Purchase Order) 정보 추출부터 공급사 납기 회신 수집까지 전 과정에서 많은 시간과 노력이 소요됩니다. 특히 PND(Production Need Date) 변경 사항 추적, 계약 납기 적정성 검증, 다수 공급사 납기 회신 취합 등의 업무가 반복적으로 발생합니다.

### 1.2 솔루션 개요
**AI Agent 기반 납기관리 자동화 시스템**을 통해 SCM팀의 핵심 업무를 자동화합니다. AI Agent가 실시간으로 데이터를 수집, 분석, 알림하여 담당자가 의사결정에 집중할 수 있도록 지원합니다.

### 1.3 핵심 가치
- ⏱️ **업무 시간 단축**: 반복 작업 자동화로 70% 이상 시간 절감
- 🎯 **리스크 조기 감지**: AI가 납기 위험을 사전에 파악하여 알림
- 📊 **데이터 일원화**: 분산된 납기 정보를 단일 플랫폼에서 관리
- 🤖 **AI Agent 협업**: 사람과 AI가 협력하는 새로운 업무 방식

## 4. 기능 요구사항

### 4.1 STEP ① PO 정보 추출

**기능**: GCOPS에서 PO 정보를 자동으로 추출하여 표준화된 Sheet를 생성

**결과 화면**:
- AI Agent 진행 상태 (Progress Bar + 실시간 로그)
- 추출된 데이터 테이블
- 요약 통계 (총 건수, 구분별 건수, 공급사 수)

---

### 4.2 STEP ② 계약 납기 검증

**기능**: '발주일' + 'Lead Time' 기준으로 '계약납기일' 적정성 자동 판단

**판단 로직**:
```
예상 완료일 = 발주일 + Lead Time

🔴 빨간색: 예상 완료일 > 계약납기일 (납기 지연 위험)
🟡 노란색: 계약납기일 - 예상완료일 ≤ 2일 (주의 필요)
🟢 초록색: 계약납기일 - 예상완료일 > 2일 (정상)
```

**결과 화면**:
- 검증 결과 요약 (정상/주의/위험 비율 차트)
- 위험 항목 목록 (즉시 조치 필요)
- 전체 데이터 테이블 (상태별 필터링)

---

### 4.3 STEP ③ PND 일정 변경 사항 검토

**기능**: 설계팀의 PND 변경 이력 추적 및 담당자 알림

**용어 정의**:
- **PND**: 생산에 필요한 자재 도착 기한
- **변경된 PND**: 설계팀에서 새로 지정한 PND
- **PND 변경**: 변경이 발생한 날짜

**결과 화면**:
- 변경 현황 요약 (변경 건수, 앞당겨짐/늦춰짐 비율)
- 최근 변경 알림 카드 (NEW 표시)
- 변경 이력 타임라인

---

### 4.4 STEP ④ 생산 보급 요청일 확인

**기능**: 생산팀 보급 요청 자동 수신 및 Sheet 업데이트

**플로우**:
```
[생산팀 시스템] → [AI Agent] → [표준화 Sheet] → [담당자 알림]
```

**결과 화면**:
- 보급 요청 현황 (긴급/일반/미요청)
- 신규 보급 요청 알림 카드
- 자동 업데이트 로그

---

### 4.5 STEP ⑤ 자재별 납기일 적정성 판단

**기능**: 계약납기일 대비 보급요청일 비교 분석

**판단 로직**:
```
🔴 빨간색: 계약납기일 > 보급요청일 (보급 불가능)
🟡 노란색: 보급요청일 - 계약납기일 ≤ 2일 (촉박)
🟢 초록색: 보급요청일 - 계약납기일 > 2일 (여유)
```

**결과 화면**:
- 적정성 분석 결과 차트
- 위험 항목 상세 목록

---

### 4.6 STEP ⑥ 공급사 납기 계획 요청

**기능**: AI Agent가 공급사에게 납기 예정일 요청 메일 자동 발송

**결과 화면**:
- 메일 발송 현황 테이블 (공급사별 상태)
- 메일 미리보기
- 발송 진행률

---

### 4.7 STEP ⑦ 공급사 납기 회신 수집

**기능**: 공급사 납기 회신 자동 수집 및 통합

**결과 화면**:
- 회신 현황 대시보드 (제출률, 공급사별 상태)
- AI Agent 데이터 통합 로그
- 미제출 공급사 리마인더 예정

---

### 4.8 STEP ⑧ 납기 비교 분석

**기능**: 1차/2차/3차 납기 변동 추적, 위험 항목 알림

**결과 화면**:
- 납기 변동 현황 요약
- 위험 항목 알림 카드 (권장 조치 포함)
- 납기 변동 추이 차트
